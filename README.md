## Setup

### 1. Clone the repository

```bash
git clone <repo-url>
cd BLYS_Task
```

### 2. Create and activate a virtual environment

```bash
python -m venv venv
# Windows
venv\Scripts\activate
# macOS / Linux
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Download spaCy model

```bash
python -m spacy download en_core_web_sm
```

### 5. Add the dataset

Place your `customer_data.csv` file inside the `data/` directory.

---

## Execution

### Step 1 — Run the notebooks (Sections 1, 2, 3)

Run each notebook in order. Each is self-contained and sets its own working directory.

```bash
# Option A — open individually
jupyter notebook notebooks/section1_customer_analysis.ipynb
jupyter notebook notebooks/section2_recommendation_model.ipynb
jupyter notebook notebooks/section3_chatbot.ipynb

# Option B — open the index and follow links
jupyter notebook notebook.ipynb
```

| Notebook | Output |
|----------|--------|
| `section1_customer_analysis.ipynb` | `reports/customer_analysis.md` + 4 plot PNGs |
| `section2_recommendation_model.ipynb` | `models/recommendation_model.pkl` |
| `section3_chatbot.ipynb` | `models/chatbot_model.pkl` |

### Step 2 — Start the API (Section 4)

```bash
python api.py
```

The API will be available at `http://localhost:8000`.

Interactive Swagger docs: `http://localhost:8000/docs`

## API Endpoints

### `POST /recommend`

Returns recommended services for a given customer.

**Request:**
```json
{
  "customer_id": 1001
}
```

**Response:**
```json
{
  "customer_id": 1001,
  "recommended_services": ["Massage", "Wellness Package", "Facial"],
  "confidence_scores": {
    "Massage": 0.72,
    "Wellness Package": 0.18,
    "Facial": 0.10
  }
}
```

---

### `POST /chatbot`

Processes a natural language customer query and returns an AI-generated response.

**Request:**
```json
{
  "session_id": "abc-123",
  "user_message": "Can I reschedule my booking?"
}
```

**Response:**
```json
{
  "session_id": "abc-123",
  "bot_response": "Sure, I can help with rescheduling. Could you please provide your booking ID?",
  "intent": "reschedule",
  "intent_confidence": 0.94,
  "current_state": "IDLE"
}
```

---

### `GET /health`

```json
{
  "status": "ok",
  "recommendation_model": true,
  "chatbot_model": true
}
```
